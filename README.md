# Desafio Técnico – Ciência de Dados

Este repositório contém o notebook desenvolvido para o **Desafio Técnico de Ciência de Dados**
do processo seletivo.

O objetivo do projeto foi integrar dados provenientes de múltiplas fontes,
realizar limpeza e padronização, explorar padrões operacionais e comparar
o desempenho entre unidades empresariais.

---

## Conteúdo do Notebook

O notebook inclui as seguintes etapas:

- Integração de dados em diferentes formatos (CSV, TXT, JSON e bancos SQLite)
- Tratamento de inconsistências, dados faltantes e padronização de colunas
- Criação de variável temporal com precisão de microssegundos
- Análise exploratória dos dados
- Comparação de performance entre Unidade A e Unidade B
- Identificação de subcategorias problemáticas (gargalos operacionais)
- Análises temporais (picos e comportamento por hora do dia)
- Visualizações estáticas e interativas
- Conclusões baseadas em evidências
